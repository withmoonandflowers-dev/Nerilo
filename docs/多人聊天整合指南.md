# 多人聊天整合指南

## 概述

本指南說明如何將 `MultiP2PManager` 整合到 `ChatPage` 中，以支援多人聊天功能。

## 架構說明

### 星型拓撲

- **房主（Host）**：作為中心節點，與所有其他用戶建立連線
- **其他用戶**：只與房主建立連線
- **訊息路由**：
  - 非房主發送：User -> 房主 -> 其他所有用戶
  - 房主發送：房主 -> 所有用戶（並行）

### 連線管理

- 每個連線使用獨立的 DataChannel（label: `chat-{remoteUid}`）
- 房主管理所有連線的狀態
- 自動處理新用戶加入和離開

const initializeP2P = async (room: P2PRoom, overrideParticipantCount?: number) => {
  const effectiveParticipantCount = overrideParticipantCount ?? room.participants.length;
  const isHost = room.ownerUid === uid;
  
  // 決定使用多人模式還是雙人模式
  const shouldUseMultiP2P = effectiveParticipantCount > 2;
  useMultiP2P.current = shouldUseMultiP2P;
  
  if (shouldUseMultiP2P) {
    // 使用 MultiP2PManager
    await initializeMultiP2P(room, isHost);
  } else {
    // 使用現有的 P2PManager（2 人聊天）
    await initializeSingleP2P(room, isHost);
  }
};

### 步驟 4：實作多人初始化

```typescript
const initializeMultiP2P = async (room: P2PRoom, isHost: boolean) => {
  if (multiP2PManagerRef.current) return;
  
  const multiP2P = new MultiP2PManager(roomId, uid, isHost);
  multiP2PManagerRef.current = multiP2P;
  
  if (isHost) {
    // 房主：與所有其他用戶建立連線
    for (const participantUid of room.participants) {
      if (participantUid !== uid) {
        try {
          await multiP2P.addConnection(participantUid);
        } catch (error) {
          console.error('[ChatPage] Failed to add connection', {
            roomId,
            remoteUid: participantUid,
            error,
          });
        }
      }
    }
  } else {
    // 非房主：只與房主建立連線
    try {
      await multiP2P.addConnection(room.ownerUid);
    } catch (error) {
      console.error('[ChatPage] Failed to add connection to host', {
        roomId,
        hostUid: room.ownerUid,
        error,
      });
    }
  }
  
  // 監聽訊息
  multiP2P.onMessage((message) => {
    setMessages((prev) => {
      if (prev.some((m) => m.messageId === message.messageId)) {
        return prev;
      }
      return [...prev, message];
    });
  });
  
  // 監聽連線狀態
  const connectionStates = multiP2P.getConnectionStates();
  // 更新整體連線狀態（例如：如果所有連線都 connected，則顯示 connected）
  const allConnected = Array.from(connectionStates.values()).every(
    state => state === 'connected'
  );
  setConnectionState(allConnected ? 'connected' : 'connecting');
};
```

### 步驟 5：處理參與者變化

```typescript
unsubscribeRoomRef.current = RoomService.subscribeRoom(roomId, async (updatedRoom) => {
  // ... 現有的驗證邏輯 ...
  
  // 如果使用多人模式，處理參與者變化
  if (useMultiP2P.current && multiP2PManagerRef.current) {
    const multiP2P = multiP2PManagerRef.current;
    const isHost = updatedRoom.ownerUid === uid;
    const currentParticipants = updatedRoom.participants;
    const lastParticipants = lastParticipantListRef.current || [];
    
    // 檢測新加入的用戶
    const added = currentParticipants.filter(uid => !lastParticipants.includes(uid));
    for (const participantUid of added) {
      if (isHost && participantUid !== uid) {
        // 房主：與新用戶建立連線
        try {
          await multiP2P.addConnection(participantUid);
          console.log('[ChatPage] Added connection to new participant', {
            roomId,
            remoteUid: participantUid,
          });
        } catch (error) {
          console.error('[ChatPage] Failed to add connection to new participant', {
            roomId,
            remoteUid: participantUid,
            error,
          });
        }
      }
    }
    
    // 檢測離開的用戶
    const removed = lastParticipants.filter(uid => !currentParticipants.includes(uid));
    for (const participantUid of removed) {
      try {
        await multiP2P.removeConnection(participantUid);
        console.log('[ChatPage] Removed connection to participant', {
          roomId,
          remoteUid: participantUid,
        });
      } catch (error) {
        console.error('[ChatPage] Failed to remove connection', {
          roomId,
          remoteUid: participantUid,
          error,
        });
      }
    }
    
    lastParticipantListRef.current = currentParticipants;
  }
});
```

### 步驟 6：修改訊息發送邏輯

```typescript
const handleSend = async () => {
  if (!inputValue.trim()) return;
  
  const message: ChatMessage = {
    messageId: generateUUID(),
    from: uid,
    content: inputValue.trim(),
    timestamp: Date.now(),
  };
  
  try {
    if (useMultiP2P.current && multiP2PManagerRef.current) {
      // 使用多人模式
      await multiP2PManagerRef.current.sendMessage(message);
    } else if (chatServiceRef.current) {
      // 使用雙人模式（現有邏輯）
      await chatServiceRef.current.sendMessage(inputValue.trim());
    }
    
    setInputValue('');
    
    // 添加到本地訊息列表
    setMessages((prev) => [...prev, message]);
  } catch (error) {
    console.error('Error sending message:', error);
  }
};
```

## 注意事項

### 1. 向後兼容

- 保持 2 人聊天使用現有的 `P2PManager`
- 只在參與者數量 > 2 時使用 `MultiP2PManager`

### 2. 連線建立時序

- 房主需要等待所有參與者都加入後再建立連線
- 新用戶加入時，房主需要主動建立連線

### 3. 連線狀態顯示

- 多人模式下，需要綜合所有連線的狀態
- 建議顯示：`3/5 已連線` 或 `連線中...`

### 4. 效能考量

- 建議最大用戶數：10-15 人
- 監控每個連線的狀態
- 自動重連失敗的連線

## 測試建議

1. **2 人聊天**：確保向後兼容
2. **3 人聊天**：測試基本多人功能
3. **5 人聊天**：測試效能和穩定性
4. **用戶加入**：測試動態添加連線
5. **用戶離開**：測試動態移除連線
6. **連線失敗**：測試重連機制
