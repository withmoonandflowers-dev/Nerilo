## 🚀 快速開始指南

### 1. 安裝依賴
```bash
npm install
cd functions
npm install
cd ..
```

### 2. 設定環境變數
複製 `env.example.txt` 為 `.env.local` 並填入 Firebase 配置：
```bash
cp env.example.txt .env.local
# 編輯 .env.local 填入實際值
```

### 3. 初始化 Firestore
在 Firebase Console 中：
1. 建立 `features/chat` 文件（參考上方）
2. 建立測試使用者並設定 role（透過 Cloud Functions `setRole`）

### 4. 啟動開發伺服器
```bash
npm run dev
```

### 5. 部署到 Firebase
```bash
# 部署 Firestore Rules
firebase deploy --only firestore:rules

# 部署 Functions
firebase deploy --only functions

# 部署 Hosting
npm run build
firebase deploy --only hosting
```

## 📝 注意事項

1. **Firestore Security Rules**：已修復 participants 陣列檢查邏輯
2. **P2P 連線**：需要兩個使用者同時在線才能建立連線
3. **資料隱私**：所有聊天內容僅透過 P2P 傳輸，不儲存在 Firestore
4. **IndexedDB**：聊天紀錄儲存在瀏覽器本機，清除瀏覽器資料會遺失
