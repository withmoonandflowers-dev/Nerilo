## 4. 若無法刪除時

若在檔案總管或 IDE 中無法刪除某檔案（權限或鎖定），請：

1. 關閉佔用該檔的程式後再刪除，或
2. 以**系統管理員**身分執行 `scripts\cleanup-removed-docs.bat`（腳本僅刪除列出的 docs 與 dummy，不影響保留文件）

清理完成後，文件以本說明與 `docs/README.md` 為準。
